## 这本书解决什么问题

很多人保存了大量 PDF、网页、视频和笔记，真正需要时依然找不到、核验不了、复用不起来。

本书围绕来源工程、证据核验和真实交付场景，讲清楚如何用 NotebookLM 完成行业研究、论文阅读、会议沉淀、课程准备和内容生产。

## 内容概览

全书约 7.4 万字，包含 23 个正式章节与 5 个附录，覆盖：

- NotebookLM 的定位、能力边界和工具分工
- 来源导入、组织、核验、隐私与维护
- 行业研究、文献综述、市场分析和深度阅读
- 会议、学习、演讲与培训工作流
- 内容知识库、选题、旧内容复用和生产链
- 40 个职业场景与 50 条实战 Prompt
